# Bank_Client_dashboard
"A comprehensive dashboard that provides insights into customer demographics, account balances, loans, deposits, credit card usage, and overall client engagement to support data-driven banking decisions."

Professional Description:-

"The Bank Client Dashboard offers a centralized view of customer banking activities, including account balances, savings, loans, deposits, and credit card transactions. It helps identify customer trends, monitor financial performance, and support strategic decision-making through interactive visualizations and key performance indicators (KPIs)."

Data Analyst Portfolio Description:-

"Designed an interactive Bank Client Dashboard using Power BI to analyze customer demographics, account holdings, loan portfolios, deposits, and transaction behavior. The dashboard enables stakeholders to track key banking metrics, identify high-value customers, and uncover business insights through dynamic filtering and visualization."

Dashboard KPIs:--
Total Clients
Total Deposits
Total Loans
Credit Card Balance
Average Client Balance
Client Distribution by Gender
Client Distribution by Age Group
Loan vs Deposit Analysis
Account Type Breakdown
Top Customers by Balance

#One-Line Description--

"Transforming banking data into actionable customer insights through interactive analytics and visualization."
